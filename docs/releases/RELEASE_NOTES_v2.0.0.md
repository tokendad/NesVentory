# NesVentory v2.0.0

Release tag: https://github.com/tokendad/NesVentory/releases/tag/v2.0.0

## Summary
v2.0.0 is a prior major release that established many core features and architecture improvements still used today. This release introduced foundational APIs, persistent storage patterns, and the first Docker-based deployment artifacts.

## Highlights
- Introduced the core API for managing items, locations, and tags.
- Added attachments support and basic preview handling.
- First official Docker image and Compose setup for easier deployments.
- Basic CSV import/export and backup tooling.

## What's new (at the time)
- Core CRUD for items and locations with validation and authentication.
- Attachments support for images and documents.
- Initial CSV import/export and backup endpoints.
- Docker and Compose deployment support.

## Improvements & fixes
- Improved persistence and stability compared to earlier v1 releases.
- Fixed multiple reliability issues in the import pipeline and attachment handling.

## Notes
- This is a historical release. If you are upgrading from v2.x to any v3.x or v4.x release, consult the current release notes for compatibility and migration guidance.
