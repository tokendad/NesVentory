# NesVentory v4.8.1

Release tag: https://github.com/tokendad/NesVentory/releases/tag/v4.8.1

## Summary
This release introduces the new video upload feature for locations, allowing users to attach videos to location entries. Additionally, it includes upgrades to packages.

**Python version downgraded:** Python was downgraded from 3.14 to 3.11 due to stability issues encountered with 3.14. Users should ensure their environments are updated accordingly, as this is a significant operational change.
## New Features
- Added video upload support for locations: Users can now upload and attach videos to location entries, enhancing the ability to document and share rich media content.
- Upgrades to packages
